# 宣盟获客诊断 Agent

这是一个可上线交付的外贸独立站获客诊断 Agent。

## 核心能力

- 输入客户官网，服务端真实抓取首页、产品页、About、Contact、FAQ/Blog 等页面
- 自动提取标题、描述、H1、页面文字、联系方式、询盘入口、工厂/资质/案例/FAQ 信号
- 录入或自动回填客户公司、官网、产品、目标市场、渠道和经营数据
- 按官网承接、SEO、GEO、渠道自主权、询盘转化五个维度评分
- 自动识别获客瓶颈和优先级问题
- 生成 SEO/GEO 内容计划、关键词方向、90 天路线图和销售跟进话术
- 支持复制报告、打印/导出 PDF、保存本地客户档案、导入/导出 JSON 档案

## 使用方式

### 后端上线版

```bash
npm start
```

默认访问：

```text
http://127.0.0.1:3000
```

建议流程：

1. 填入客户官网或产品页链接
2. 点击“智能识别并填入”
3. 后端会抓取官网页面并自动回填诊断字段
4. 点击“生成诊断”
5. 到“交付报告”复制、打印或导出方案
6. 点击“保存”沉淀客户档案

### 纯静态备用

直接打开 `index.html` 也能使用，但无法真实抓取官网，只能根据粘贴内容做本地推断。

## 交付边界

当前后端 MVP 不接付费 SEO 数据源，先基于官网真实页面内容做规则诊断。它适合销售初诊、客户方案演示、轻量交付。

后续可升级为：

- 接入真实 AI 模型
- 连接 Google Search Console / GA4
- 连接 Ahrefs / Semrush / Similarweb / PageSpeed Insights
- 生成 Word/PDF 正式报告
- 多用户账号与客户权限管理
